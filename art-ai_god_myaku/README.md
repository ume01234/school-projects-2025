# AI God - Mandala

2025年大阪万博のキャラクター「ミャクミャク」と「曼荼羅」を融合させたジェネラティブアート作品。

聖書（筐体）の中に埋め込まれたスマホで動作するWebアプリ。インカメラで鑑賞者を検知し、p5.jsで描かれた幾何学的な「目」の集合体がインタラクティブに反応する。

## 技術スタック

- **HTML5, CSS3, JavaScript**
- **p5.js** - グラフィック描画（WEBGLモード）
- **p5.sound** - 音響制御
- **ml5.js** - 顔認識（MediaPipe Face Mesh）

## 機能

- **明るさ検知**: カメラ映像の平均輝度を計測し、CLOSED/OPENモードを自動切り替え
- **顔認識**: ml5.jsで顔をトラッキングし、全ての目が顔の位置に追従
- **サウンド制御**: OPENモード時に音声をループ再生
- **フラクタル配置**: 再帰的アルゴリズムによる円状配置
- **順次表示**: 本を開いた時、中心から外側に向かって目が順に現れる
- **3D回転**: 画面全体が時計回りにゆっくり回転

## セットアップ

1. リポジトリをクローン
2. VS Codeでプロジェクトを開く
3. Live Server拡張機能で`index.html`を起動
4. ブラウザでカメラへのアクセス許可を付与
5. ページをクリックしてから本を開く（音声再生のため）

## 操作方法

- **本を閉じる**: カメラを手で覆って暗くする → CLOSEDモード（画面が黒、音声停止）
- **本を開く**: 手を離して明るくする → OPENモード（目が表示、音声ループ再生）

## デバッグ機能

- **Dキー**: 顔パーツのデバッグポイント表示の切り替え
- **Tキー**: 明るさの閾値を50↔100で切り替え
- **Sキー**: 現在の画面を画像として保存

## 作品の特徴

- **ミャクミャクをイメージした眼球**: 紫の虹彩と青い瞳孔を持つ3Dの目
- **フラクタル配置**: 中心から再帰的に広がる円状配置
- **インタラクティブ**: 顔の動きに応じて全ての目が追従
- **アニメーション**: 画面全体が時計回りに回転、瞳孔が生きているように動く
