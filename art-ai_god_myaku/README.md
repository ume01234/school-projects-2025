# AI God - Mandala

2025年大阪万博のキャラクター「ミャクミャク」と「曼荼羅」を融合させたジェネラティブアート作品。

聖書（筐体）の中に埋め込まれたスマホで動作するWebアプリ。インカメラで鑑賞者を検知し、p5.jsで描かれた幾何学的な「目」の集合体がインタラクティブに反応する。

## 技術スタック

- **HTML5, CSS3, JavaScript**
- **p5.js** - グラフィック描画
- **p5.sound** - 音響制御
- **ml5.js** - 顔認識（MediaPipe Face Mesh）

## 機能

- **明るさ検知**: カメラ映像の平均輝度を計測し、CLOSED/OPENモードを自動切り替え
- **顔認識**: ml5.jsで顔をトラッキングし、視覚要素が顔の位置に追従
- **サウンド制御**: OPENモード時に音声再生（現在はダミー処理）

## セットアップ

1. リポジトリをクローン
2. VS Codeでプロジェクトを開く
3. Live Server拡張機能で`index.html`を起動
4. ブラウザでカメラへのアクセス許可を付与

## プロジェクト構成

```
├── index.html          # エントリーポイント
├── style.css           # スタイル定義
├── js/
│   └── sketch.js       # メインロジック
├── assets/
│   ├── sounds/         # 音声ファイル用
│   └── images/         # 画像ファイル用
└── docs/
    └── bible_prompt.md # 制作過程の記録
```

## デバッグ機能

- **Dキー**: 顔パーツのデバッグポイント表示の切り替え
- **Tキー**: 明るさの閾値を50↔100で切り替え
