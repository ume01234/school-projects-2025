# 目隠しリバーシ

盤面の一部が見えない状態でプレイする自作オセロ（リバーシ）アプリ。認知負荷が高い状況下でのゲームプレイを体験できます。

## 概要

通常のオセロとは異なり、盤面情報が制限された状態でAIと対戦します。プレイヤーは黒（先攻）を担当し、見えない部分の石の配置を記憶・推測しながらプレイする必要があります。

## ゲームモード

3段階の難易度から選択できます。

| モード | 難易度 | 説明 |
|--------|--------|------|
| blindfold1 | 軽度 | 外周（1列目と8列目、A列とH列）が見えない |
| blindfold2 | 中程度 | 盤面の半分がランダムに見えない（ゲーム開始時に決定） |
| blindfold3 | 重度 | 中央4マス（d4, d5, e4, e5）以外すべて見えない |

## ルール

- プレイヤーは黒、AIは白
- 合法手（石を置ける場所）は表示されない
- 任意の空きマスをクリックできる
- 不正な手（実際には置けない場所）を選ぶとパス扱いになり「Miss!」と表示される
- AIの直前の手は赤枠で表示される
- 両者とも置ける場所がなくなるとゲーム終了

## 技術スタック

- React 18
- TypeScript
- Vite

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build
```
