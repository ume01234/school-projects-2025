# 目隠しオセロ

盤面情報を制限した「目隠しオセロ」による認知負荷とUXを研究するWebプラットフォーム

## ゲームモード

- **軽度（blindfold1）**: 外周が見えない
- **中程度（blindfold2）**: 盤面の半分がランダムに見えない
- **重度（blindfold3）**: 中央4マス以外見えない

## ルール

- 合法手は表示されない
- 任意の空きマスに置ける
- 不正な手（合法手以外）はパス扱い
- AIの直前の手は赤枠で表示

## 技術スタック

- React + TypeScript + Vite

## セットアップ

```bash
npm install
npm run dev
```
