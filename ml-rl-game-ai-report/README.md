# 目隠しリバーシ

2025秋~の情報メディア実験Bの成果物リポジトリ
盤面の一部が見えない状態でプレイする自作オセロ（リバーシ）アプリ。認知負荷が高い状況下でのゲームプレイを体験できます。

## 概要

通常のオセロとは異なり、盤面情報が制限された状態でAIと対戦します。プレイヤーは黒（先攻）を担当し、見えない部分の石の配置を記憶・推測しながらプレイする必要があります。

## ゲームモード

3段階の難易度から選択できます。

| モード | 難易度 | 説明 |
|--------|--------|------|
| blindfold1 | 軽度 | 外周（1列目と8列目、A列とH列）が見えない |
| blindfold2 | 中程度 | 盤面の半分がランダムに見えない（ゲーム開始時に決定） |
| blindfold3 | 重度 | 中央4マス（d4, d5, e4, e5）以外すべて見えない |

## ルール

- プレイヤーは黒、AIは白
- 合法手（石を置ける場所）は表示されない
- 任意の空きマスをクリックできる
- 不正な手（実際には置けない場所）を選ぶとパス扱いになり「Miss!」と表示される
- AIの直前の手は赤枠で表示される
- 両者とも置ける場所がなくなるとゲーム終了

## 技術スタック

- React 18
- TypeScript
- Vite

## ファイル構成

```
├── index.html              # エントリーポイント
├── vite.config.ts          # Vite設定
├── src/
│   ├── main.tsx            # Reactアプリのマウント
│   ├── App.tsx             # メインコンポーネント（ゲーム状態管理）
│   ├── App.css             # アプリ全体のスタイル
│   ├── index.css           # グローバルスタイル
│   ├── components/         # UIコンポーネント
│   │   ├── Board.tsx       # 盤面コンポーネント
│   │   ├── Cell.tsx        # 各マスのコンポーネント
│   │   ├── GameInfo.tsx    # ターン・スコア表示
│   │   ├── GameResult.tsx  # 結果画面
│   │   ├── ModeSelector.tsx # モード選択画面
│   │   └── *.css           # 各コンポーネントのスタイル
│   └── game/               # ゲームロジック
│       ├── types.ts        # 型定義
│       ├── othelloLogic.ts # オセロのルール実装
│       ├── randomAI.ts     # ランダムAI
│       └── visibilityUtils.ts # 盤面可視性の制御
└── dist/                   # ビルド成果物（自動生成）
```
